For each potential top-left corner `(row, col)` in `array1`: * Extract the submatrix from `array1` starting at `(row, col)` with the same dimensions as `array2`. * Compare this extracted submatrix element-by-element with `array2`. * If they match completely, we've found a match and can return `(row, col)`.
